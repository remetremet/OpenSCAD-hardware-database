# OpenSCAD-hardware-database

ISO/EN/DIN precise database for OpenSCAD

 - washers
 - nuts
 - bolts
 - screws
 - ball bearings
 - linear bearings
 - axial bearings
 - axial fans
 - radial fans
 - belts
 - 3D printer heatbeds
 - DC and stepper motors
